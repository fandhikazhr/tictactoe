# Tictactoe 

Hello (ʘ‿ʘ)╯

TicTacToe Games on your's terminal ❌⭕❌

<h3> Usage </h3>

First compile file `tictactoe.cpp`

```
clang++ -Wall -std=c++11 tictactoe.cpp -o tictactoe
```

## How to play

Select the column number you want to fill

1️⃣ 2️⃣ 3️⃣

4️⃣ 5️⃣ 6️⃣

7️⃣ 8️⃣ 9️⃣

- <kbd>X</kbd> is Player 1
- <kbd>O</kbd> is Player 2
